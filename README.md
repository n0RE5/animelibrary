# DESCRIPTION
This is fullstack REACT[frontend] and NodeJS[backend] Anime Website with JTW based AUTH system, RATING system, FAVORITE system and etc.
Project technologies [PERN stack]:
# Frontend:
    - React
    - Redux
    - TypeScript
# Backend:
    - NodeJS
    - PostgreSQL
    - ExpressJS
    (NodeJS libraries)
    - JWT
    - BCrypt
    - Sequelize
    - cors
# START SERVER
### `npm run dev`
# START CLIENT
### `npm start`
# DEPENDS
Setting UP:

You will have to install NodeJS & React deps so after installation run

### `cs client`
### `npm i`

### `cd server`
### `npm i`

Install and configure PostgreSQL
Install NodeJS
Edit .env files both client and server side on your own

!!! Important !!! Change secret key in .env server file AND MAKE IT SECRET PLEASE!
!!! Important !!! Create folder static in server folder
Now your server fully configured and ready for users wave.
